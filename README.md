## Setup

**Step 1:**  Clone repository

```sh
 git clone https://github.com/kirvinod/mybbcPush.git
```

**Step 2:** Go to root directory

```sh
cd mybbcPush/
```

**Step 3:** Run application

```sh
mvn spring-boot:run
```

**Step 4:** Application by default will be serving on port 8080

```sh
http://localhost:8080
```
